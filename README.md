# Level-REGEX
Regex shinyliveApp and library to explain the clinician's made Regular Expression Extractions of the Levetiracetam vs Lacosamide comparison in Trauma ICU university hospital of Rennes

This project coded in R mostly by vibe coding with claude but manually modified and readed, consisted in doing a comparison between Levetiracetam (the common practice) and Lacosamide (new practice) for ensuring the safety of this last molecule.
Data were extracted from the local Electronic Data Wharehouse named Ehop. We notably used ReGex as main NLP method to extract relevant datas on patients comorbidity and clinical variables.

It is composed of :
The shiny live application to visualise the full Regex Work.
And the distincts codes for preparing and extracting the informationsfrom medicals observations.
